# Flux d'affichage des pages d'erreur dynamiques avec Flask et Jinja2

```text
Utilisateur
    │
    ▼
Accède à une page ou action
    │
    ▼
Flask détecte l'erreur
(404, 500, etc.)
    │
    ▼
Gestionnaire d'erreur Flask
@app.errorhandler(code)
    │
    ▼
Récupération des infos d'erreur
(erreur.code, erreur.message)
    │
    ▼
Passage des variables à Jinja2
render_template("erreur.html", erreur=erreur)
    │
    ▼
Template Jinja2 (erreur.html)
 ┌───────────────────────────────┐
 │ <h1>Erreur {{ erreur.code }}</h1>  │
 │ <p>{{ erreur.message }}</p>        │
 │ <img src="{{ url_for('static',      │
 │ filename='images/' + erreur.code|string + '.png') }}" alt="Erreur {{ erreur.code }}"> │
 └───────────────────────────────┘
    │
    ▼
Jinja2 compile le HTML dynamique
    │
    ▼
HTML complet envoyé au navigateur
    │
    ▼
Affichage pour l'utilisateur
 ┌───────────────────────────────┐
 │ - Code et message corrects    │
 │ - Image spécifique à l'erreur│
 │ - Structure HTML intacte      │
 └───────────────────────────────┘
